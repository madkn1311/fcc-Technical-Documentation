Introduction

Markdown is a lightweight markup language that was created by John Gurber and Aaron Swartz in 2004. The reason to create such a markup language was to make it easier for humans to read it in its source code form. Markdown is widely used today for documentation, blogging, online forums, readme files and many other purposes.

Other markup languages like HTML, XML have a multitude of tags for formatting text which are easy for a computer to understand, but that is not the case for human readers as it is difficult to understand how the text will appear after the computer renders it, so these languages are not very user-friendly and require beforehand experience.

On the other hand, markdown makes formatting text for web pages easier as its tags are simpler than that of HTML and they get converted to HTML automatically so there is no need to know html to write for web pages. It doesn't cover all html tags but the most common formatting options.

Headings

To create a heading use the # sign before a word or phrase. There are six heading levels, so use the corresponing number of # signs to create different heading levels.

Syntax: 

 
             
           # Level 1
           
           ## Level 2

Points to remember:

  • Always put space between the # sign and the heading for compatibility.
  • Always leave blank lines before and after the heading.
Paragraphs

Use a blank line to separate one or more lines of text, to create paragraphs.

Syntax: 

 
             
           I really like markdown.
           
           I think i will use it to format all of my documents from now on.

Points to remember:

  • Don't indent paragraphs with space or tabs unless they are in a list.

Bold

To make the text bold add two asterisks or underscores on either side of the word or phrase and to make a word bold in the middle of a sentence for emphasis, add two asterisks on either side of the word without spaces around the letters.

Syntax: 

 
             
           I just love **bold text**.
           
           I just love __bold text__.
           
           Love **is** bold.

Points to remember:

  • For compatibility, always use asterisks to make the word in the middle of a sentence bold.
  • This applies for italic as well as bold and italic.
Italic

To Italicize text add one asterisk or underscore on either side of the word or phrase and to italicize a word in the middle of a sentence use one asterisk on either side of the word without spaces around the letters.

Syntax: 

 
             
           Italicized text is the *cat's meow*.
           
           Italicized text is the _cat's meow_.
           
           A *cat* meows.

Blockquotes

Add a > sign before a paragraph, to create a blockquote.

Syntax: 

 
             
           > Dorothy followed her through many of the beautiful rooms in her castle.

Points to remember:

  • For compatibility, put blank lines before and after blockquotes.

Syntax: 

 
             
           1. First item
           2. Second item
           3. Third item
           4. Fourth item
           
           1. First item
           1. Second item
           1. Third item
           1. Fourth item

Unordered Lists

To create unordered lists use dashes, asterisks or plus signs before the list items.

Syntax: 

 
             
           - First item
           - Second item
           - Third item
           - Fourth item
           
           + First item
           + Second item
           + Third item
           + Fourth item

Points to remember:

  • For compatibility, don't mix and match delimiters in the same list.

Code

To denote a word or phrase as code, enclose it in backticks.

Syntax: 

 
             
           At the command prompt type `nano`.

Code Blocks

To create code blocks, indent every line of the block by atleast four spaces or one tab.

Syntax: 

 
             
                html
                      head
                    
                      head
                html

Horizontal Rule

Use three or more asterisks, dashes or underscores on a line by themselves.

Syntax: 

 
             
           ***
           
           ---
           
           ___

Points to remember:

  • For compatibility, put blank lines before and after horizontal rules.

Links

To create a link, enclose the link text in square brackets and then immediately follow it with the URL in parentheses.

Syntax: 

 
             
           My favorite search engine is [Duck Duck Go](https://duckduckgo.com).

Images

To add an image, add an exclamation mark, followed by alt text in square brackets, followed by the path or URL of the image in parntheses. You can also add a title after the URL within double quotes.

Syntax: 

 
             
          ![The San Juan Mountains are beautiful!](/assets/images/san-juan-mountains.jpg)
